Qu'est ce que le HTML?





HTML est l'acronyme de HyperText Markup Language. Il permet d'établir le squelette de notre site grace à des balises.

Pour comprendre sa structure, on peut imaginer une grande boite qui contient plusieurs boites.

Chaque balise s'utilise dans des cas spécifiques. Il faut s'assurer de prendre la meilleure balise pour chaque situation, car les balises ont une importance dans le référencement SEO de votre site.




Pour marquer un commentaire dans le code HTML il faut écrire
